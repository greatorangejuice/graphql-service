
## Description

Musify BFF (GraphQL service)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Using

```bash
Run microservices. If microservices addresses is not default update .env file.
Open playground localhost:3000 in a browser
```
